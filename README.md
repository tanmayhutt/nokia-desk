# Nokia 3310 Emulator 📱

A web-based emulator of the iconic Nokia 3310 — classic UI, playable Snake, and retro vibes built with modern web tech.

🌐 **[Live Demo →](https://tanmayhutt.github.io/nokia-desk/)**

---

## Features

- 🐍 Playable Snake game
- 📶 Live battery & signal indicators
- 🎮 Classic Nokia keypad & UI
- 🎨 3D rendering with Three.js
- ✨ Smooth animations via Framer Motion

## Tech Stack

- **React 19** — UI framework
- **Three.js / React Three Fiber** — 3D rendering
- **Framer Motion** — animations
- **Tailwind CSS** — styling
- **Vite** — build tool

## Getting Started

```bash
# Clone the repo
git clone https://github.com/tanmayhutt/nokia-desk.git
cd nokia-desk

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Deploy

```bash
npm run deploy
```

Deploys to GitHub Pages via `gh-pages`.
